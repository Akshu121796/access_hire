AccessHire ♿💼

AccessHire is an AI-powered, accessibility-first hiring platform designed to connect persons with disabilities (PwDs) to inclusive job opportunities while helping employers hire inclusively.

🚀 Features

🔐 Firebase Authentication – Secure login for candidates and employers

💼 Accessible Job Discovery – Browse inclusive job listings with accessibility tags

🧾 My Applications Dashboard – Track job application status (Applied, Interview, Selected, etc.)

🎓 Skill Training Module – Accessible learning courses with progress tracking

🎙 Voice Search – Search jobs using speech (Web Speech API)

🔊 Read Aloud Support – Text-to-speech for better accessibility

🌗 High Contrast Mode – Improved UI visibility for low-vision users

🧠 Google Technologies Used

Firebase Authentication – User authentication

Cloud Firestore – Jobs, applications, and user data

Firebase Hosting / Vercel – Deployment

Google Chrome & ChromeVox – Accessibility testing

🤖 Google AI Tools Integrated

Web Speech API (Text-to-Speech) – Read Aloud feature

Web Speech API (Speech Recognition) – Voice-based job search

🛠 Tech Stack

Frontend: Next.js (App Router), React, TypeScript

UI: Tailwind CSS, Lucide Icons

Backend: Firebase (Auth + Firestore)

Deployment: Vercel
